주민등록등본이 급하게 필요한데 주민센터까지 가기 번거로울 때가 있죠. 인터넷으로 무료 발급이 가능한데, 정부24 사이트에 들어가면 메뉴가 너무 많아서 어디로 가야 하는지 헷갈리는 분들이 많아요. 등본뿐 아니라 자주 필요한 서류들의 바로가기 링크를 한번에 정리했어요.

> **핵심 요약:** 정부24에서 인터넷 발급하면 무료예요. 카카오톡, 네이버 등 간편인증으로 로그인할 수 있어요. 24시간 이용 가능하고 즉시 발급돼요.

## 주민등록등본·초본 발급 바로가기

### 정부24 인터넷 발급 (무료)

정부24에서 카카오톡·네이버 간편인증으로 로그인하면 24시간 무료로 즉시 발급할 수 있어요.

{{< link-button url="https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=13100000015" text="정부24 등본 발급 바로가기" >}}

| 항목 | 내용 |
|------|------|
| 수수료 | **무료** (주민센터 방문 시 400원) |
| 로그인 방법 | 공동인증서, 금융인증서, 간편인증(카카오, 네이버, PASS, 토스), 모바일 신분증 |
| 처리기간 | 즉시 발급 |
| 이용시간 | 24시간 |

- PDF로 저장하거나 바로 인쇄 가능
- 등본(세대 전원 정보)과 초본(본인 1인 정보) 모두 발급 가능

## 자주 필요한 서류 바로가기 모음

등본 외에도 자주 필요한 서류들이 있어요. 매번 검색하지 않도록 한번에 정리했어요.

### 가족관계증명서

가족 구성원 정보를 확인하는 서류로, 정부24 또는 대법원 사이트에서 무료로 24시간 발급할 수 있어요.

{{< link-button url="https://www.gov.kr/main?a=AA020InfoCappViewApp&CappBizCD=97400000004" text="정부24 가족관계증명서 발급" >}}

{{< link-button url="https://efamily.scourt.go.kr/" text="대법원 전자가족관계등록시스템" >}}

- 대법원 사이트에서는 기본증명서, 혼인관계증명서, 입양관계증명서, 제적등본도 발급 가능

### 지방세 납세증명서

지방세를 체납하지 않았음을 증명하는 서류로, 대출이나 입찰 시 자주 요구돼요.

{{< link-button url="https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=13100000056" text="정부24 납세증명서 발급" >}}

- 무료, 즉시 발급

### 건축물대장 등본

부동산 거래, 대출 등에 필요한 건축물 현황 확인용 서류예요.

{{< link-button url="https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=15000000098" text="정부24 건축물대장 발급" >}}

### 토지대장 등본

토지의 소재지, 지목, 면적, 소유자 정보가 기재된 공적 장부예요.

{{< link-button url="https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=13100000026" text="정부24 토지대장 발급" >}}

### 토지이용계획확인서

해당 토지에 적용되는 도시계획, 용도지역 등을 확인할 수 있는 서류예요. 토지 매매나 건축 전에 반드시 확인하세요.

{{< link-button url="https://www.gov.kr/mw/AA020InfoCappView.do?CappBizCD=15000000013" text="정부24 토지이용계획확인서" >}}

{{< link-button url="https://www.eum.go.kr" text="토지이음 열람 바로가기" >}}

### 인감증명서 (주의: 온라인 발급 불가)

인감증명서는 **온라인 발급이 안 돼요**. 반드시 본인이 직접 주민센터나 무인발급기를 방문해야 해요.

| 항목 | 내용 |
|------|------|
| 발급 방법 | 주민센터 방문 또는 무인발급기 |
| 수수료 | 600원 |
| 주의 | 본인 직접 방문 필수, 대리 발급 불가 |

## 자주 묻는 질문 (FAQ)

### 정부24 회원가입 없이도 발급 가능한가요?

간편인증(카카오, 네이버, PASS 등)으로 비회원 로그인이 가능해요. 별도 회원가입 없이 바로 발급할 수 있어요.

### 등본과 초본의 차이가 뭔가요?

등본은 세대 전원(같은 주소에 사는 가족 전체)의 주민등록 사항이 나오고, 초본은 본인 1인의 상세 주민등록 사항(주소 변동 이력 등)이 나와요. 대부분의 경우 등본이면 충분해요.

### 해외에서도 발급 가능한가요?

네, 인터넷이 되면 해외에서도 정부24를 통해 발급 가능해요. 다만 공동인증서가 만료되지 않았는지 확인하세요. 간편인증은 국내 전화번호가 필요할 수 있어요.

### 인감증명서를 온라인으로 발급할 수 없는 이유는?

인감증명서는 법적으로 본인 직접 발급만 허용되어 있어요. 위조 방지를 위해 반드시 본인이 주민센터나 무인발급기를 방문해서 지문인식 후 발급받아야 해요.
